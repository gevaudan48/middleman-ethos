    mixins

    breakpoint 

    @mixin breakpoint($min: 0, $max: 0) { ... }

    Description

    Media Query Breakpoint The breakpoint mixin uses the same attribute names as Bootstrap's responsive classes to make using the feature as easy to remember as possible. Passing in a responsive class will add a media query and output the passed in block.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $min

    Sets the min-width or uses preset: xs, sm, md, lg

    		 Number, String 0
    $max

    Sets the max-width

    		 Number 0

    Example

    .foo { @include breakpoint(sm) {...}}
    .foo { @include breakpoint($max: 800px) {...}}

    placeholder
